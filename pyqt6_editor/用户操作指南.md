# TACZ Lua Map 编辑器用户操作指南

## 1. 界面介绍

### 1.1 主界面布局
- **左侧面板**：节点调色板，包含所有可用的节点类型
- **中间面板**：画布区域，用于创建和连接节点
- **右侧面板**：代码预览区域，实时显示生成的 Lua 代码

### 1.2 菜单栏
- **文件**：新建、保存、打开项目
- **编辑**：复制、粘贴、撤销、重做
- **工具**：生成代码、切换主题、Scratch 预览
- **帮助**：显示帮助信息

## 2. 基本操作

### 2.1 节点操作
- **创建节点**：从左侧面板拖拽节点到画布，或双击节点名称
- **移动节点**：左键拖拽节点到目标位置
- **编辑属性**：双击节点打开属性编辑面板
- **删除节点**：选中节点后按 Delete 或 Backspace 键
- **复制节点**：选中节点后按 Ctrl+C，然后按 Ctrl+V 粘贴
- **快速复制**：按住 Alt 键拖拽节点

### 2.2 连线操作
- **创建连线**：从输出端口拖拽到输入端口
- **删除连线**：双击连线或右键选择删除
- **查看连线**：鼠标悬停在连线上会高亮显示

### 2.3 画布操作
- **平移画布**：左键拖拽空白区域，或按住 Ctrl 键左键拖拽
- **缩放画布**：使用鼠标滚轮缩放，以鼠标位置为中心
- **框选节点**：左键拖拽形成矩形框选
- **添加选区**：按住 Shift 键框选，向已有选区添加节点
- **聚焦节点**：选中节点后按 F 键
- **居中显示**：按 Home 键居中显示所有节点
- **取消选中**：按 Ctrl+D 取消所有选中状态

## 3. 高级功能

### 3.1 主题切换
- **快捷键**：Ctrl+T
- **功能**：在暗色主题和亮色主题之间切换

### 3.2 代码生成
- **快捷键**：Ctrl+G
- **功能**：根据节点图生成 Lua 代码
- **自动生成**：当节点图发生变化时，会自动更新代码

### 3.3 反向解析
- **功能**：将 Lua 代码解析为节点图
- **操作**：在代码预览面板中点击"反向解析"按钮

### 3.4 撤销/重做
- **撤销**：Ctrl+Z
- **重做**：Ctrl+Y
- **功能**：恢复到之前的操作状态

### 3.5 节点搜索
- **操作**：在左侧面板的搜索框中输入关键词
- **功能**：快速找到需要的节点
- **补全**：搜索框支持自动补全功能

### 3.6 自定义代码编辑
- **操作**：双击需要输入多行代码的节点属性
- **功能**：打开多行文本编辑器，支持语法高亮和行号显示

## 6. 故障排除

### 6.1 常见问题

**问题**：节点无法连接
**解决方案**：确保端口类型匹配（EXEC 端口只能连接到 EXEC 端口），输入端口只能有一个连接

**问题**：代码生成失败
**解决方案**：检查节点之间的连接是否完整，节点配置是否正确

**问题**：节点不显示在面板中
**解决方案**：确保节点类使用了 `@register_node` 装饰器，装饰器参数正确

### 6.2 性能优化
1. **减少节点数量**：节点数量过多会影响性能，尽量使用组合节点减少节点总数
2. **合理使用连接**：过多的连接会增加渲染和计算负担，尽量减少不必要的连接
3. **使用批处理**：对于重复的操作，使用批处理节点代替多个单独节点
4. **优化自定义节点**：在自定义节点的 `execute` 方法中避免复杂计算，尽量使用简单的逻辑
5. **使用缓存**：对于计算密集型操作，考虑使用缓存机制避免重复计算

## 7. 扩展节点

直接在 `pyqt6_editor/tacz_nodes.py` 中添加自定义节点，参见下方示例。

### 7.1 创建自定义节点
```python
from pyqt6_editor.node import Node, PortType
from pyqt6_editor.registry import register_node

@register_node("自定义分类", "#FF5722", "自定义节点", "这是一个自定义节点")
class MyNode(Node):
    def _init_ports(self):
        self.add_input("输入", PortType.DATA, "number")
        self.add_output("输出", PortType.DATA, "number")

    def _get_default_config(self):
        return {"value": 100}
```
